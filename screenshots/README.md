# EduChat Screenshots

## 📱 스크린샷 가이드

### 준비사항:
1. EduChat 앱 실행
2. API 키 설정  
3. 일반 모드와 딥러닝 모드 각각 테스트

### 추천 촬영 장면:
1. **웰컴 화면** - 초기 환영 메시지
2. **일반 모드** - 간단한 질문/답변
3. **딥러닝 모드** - 사전 질문부터 최종 분석까지
4. **설정 화면** - API 키 입력
5. **다크 모드** (가능하다면)

### 팁:
- 배경을 깨끗하게 유지
- EduChat 로고가 잘 보이게 촬영
- 다양한 대화 상황 캡처
- 실제 사용 시나리오 위주로 촬영
